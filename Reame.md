My very first game using Python and Pygame (with Gemini's help!)

It features two game modes: 
**P1 vs CPU**: Single player Mode
**P1 vs P2**: Multiplayer Mode

The game includes some Brazilian clubs and two National Teams

## How to play
1. Install Python
2. Install Pygame with the command: 'pip install pygame'
3. Run game: 'python penalti.py'

# Controls
- **Arrows:** Aim and move goalkeeper
- **Space:** Shoot
- **ESC:** Return to the main menu/Paue
- **S:**: Confirm Exit (Yes)
- **N:**: Cancel Exit (No)
- **ENTER:** Skip the celebration/Continue
