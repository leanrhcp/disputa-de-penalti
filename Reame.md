# Pênaltis: Clássico Listrado ⚽

Meu primeiro jogo utilizando Python e Pygame (muita ajuda de IA).
Possui 2 modos de jogo: P1 vs P2 e P1 vs CPU
Há alguns times brasileiros e duas seleções.

## Como jogar
1. Instale o Python.
2. Instale o Pygame com o comando:
   `pip install pygame`
3. Execute o jogo:
   `python main.py`

## Controles
- **Setas:** Mirar e Mover Goleiro (no modo manual)
- **Espaço:** Chutar
- **S:** Sair / Pausar
